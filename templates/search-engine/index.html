This tutorial will show you how to make a search engine just as good as Google with only a single page of HTML! No programming is required!

Instructions

  1. Create a "boilerplate" HTML document.
  2. Add a <form> element.
  3. Set the method to get.
  4. Set the action to https://google.com/search.
  5. Add an <input> tag.
  6. Set the type to search.
  7. Set the name to q.
  8. Add a submit button.
  9. Add extra markup if desired.
  10. Add CSS if desired.
  11. Congratulations! You now have an HTML-only (or HTML-and-CSS-only) search engine that's just as good as Google!
